# Form-field type code (integer enum)

Returns the raw `FPDF_FORMFIELD_*` integer.

## Usage

``` r
pdf_form_field_type_code(field)
```

## Arguments

- field:

  A `pdfium_form_field` handle from
  [`pdf_form_fields()`](https://humanpred.github.io/rpdfium/dev/reference/pdf_form_fields.md).

## Value

Integer scalar.
