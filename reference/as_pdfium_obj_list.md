# Coerce input to a `pdfium_obj_list`

Reverse companion to
[`as_tibble.pdfium_obj_list()`](https://humanpred.github.io/rpdfium/reference/as_tibble.pdfium_obj_list.md).

## Usage

``` r
as_pdfium_obj_list(x)
```

## Arguments

- x:

  Either a `pdfium_obj_list`, a list of `pdfium_obj` handles, or a
  tibble with a `handle` list-column.

## Value

A `pdfium_obj_list`.
