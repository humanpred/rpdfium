# Annotation `/Subj` subject text

Returns the annotation's `/Subj` subject string.

## Usage

``` r
pdf_annot_subject(annot)
```

## Arguments

- annot:

  A `pdfium_annot` handle from
  [`pdf_annotations()`](https://humanpred.github.io/rpdfium/reference/pdf_annotations.md).

## Value

Character scalar.
