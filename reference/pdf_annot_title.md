# Annotation `/T` title (author) text

Returns the annotation's `/T` title (commonly the author name).

## Usage

``` r
pdf_annot_title(annot)
```

## Arguments

- annot:

  A `pdfium_annot` handle from
  [`pdf_annotations()`](https://humanpred.github.io/rpdfium/reference/pdf_annotations.md).

## Value

Character scalar.
