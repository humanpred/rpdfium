# AGENTS.md

The working conventions for AI coding agents in this repository live in
[`CLAUDE.md`](https://humanpred.github.io/rpdfium/CLAUDE.md). Open that
file first — it covers package identity, layering, memory model, testing
under parallelism, code style, which files must be updated together, the
read-only `gh` CLI constraint, and pre-commit setup.

This file exists only so agents that follow the
[AGENTS.md](https://agents.md/) convention discover the conventions
without having to know to look for `CLAUDE.md` specifically.
